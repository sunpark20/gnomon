## 설명 (한국어)

MacBook의 조도 센서를 이용해 외장 모니터의 밝기를 자동으로 조절합니다.

MacBook 내장 화면은 주변 밝기에 따라 알아서 밝아지고 어두워지지만, 외장 모니터는 항상 같은 밝기입니다. 해가 뜨면 외장 모니터만 어둡게 보이고, 밤에는 외장 모니터만 눈부십니다.

Century Iris는 이 문제를 해결합니다.

**주요 기능**

- 주변 밝기를 실시간으로 감지하여 외장 모니터 밝기를 자동 조절
- 급격한 조도 변화(조명 켜기/끄기 등)는 즉시 반영
- 로그 곡선 기반의 자연스러운 밝기 매핑
- 메뉴바에 상주하며 현재 조도와 밝기 상태를 한눈에 확인
- 단축키로 수동 밝기 조절 가능

**이런 분께 추천합니다**

- MacBook에 외장 모니터를 연결해서 쓰는 분
- 모니터 밝기를 매번 수동으로 바꾸는 게 귀찮은 분
- 가볍고 단순한 자동 밝기 앱을 원하는 분

가입 없음. 광고 없음. 구독 없음.

## Description (English)

Century Iris uses your MacBook's ambient light sensor to automatically adjust external monitor brightness.

Your MacBook screen adapts to surrounding light on its own — but your external monitor stays the same brightness no matter what. In daylight it looks too dark; at night it's blinding.

Century Iris fixes this.

**Key Features**

- Detects ambient light in real time and adjusts external monitor brightness automatically
- Responds instantly to sudden lighting changes (lights on/off, curtains, etc.)
- Smooth, natural brightness mapping based on a logarithmic curve
- Lives in the menu bar — see current ambient light and brightness at a glance
- Manual brightness override via keyboard shortcuts

**Who is this for?**

- MacBook users with an external monitor
- Anyone tired of manually adjusting monitor brightness throughout the day
- People who want a simple, lightweight auto-brightness app

No account required. No ads. No subscription.

## 심사 메모 (App Review Notes)

> App Store 제출 시 심사팀에 보내는 설명.

This app reads the MacBook's ambient light sensor to automatically adjust screen brightness for a comfortable viewing experience. It uses only public macOS APIs (IOKit registry property reading for ambient light, CoreGraphics gamma table APIs for brightness adjustment). No private APIs are used. The app runs as a menu bar utility and requires no special permissions.
